# Idea Informatica Mia

## Progetto Informatica: Sistema Bancario in C. 

## Cartelle
### Client>
- Vue + Bootstrap 4 (4 pagine, 1 login/signup, 1 che include 3 per ogni sezione, 1 menu operazioni, 2 chat bot AI per riferimenti e aiuto help e 3 pagina profilo utente e impostazioni). 

### Server>
- NodeJS + Express (faranno da ponte per il Core in C, richieste HTTP e altro, e avranno riferimento API per il chat bot AI (gemini 3.0 flash) che non fa parte del C). 

### Core>
- C, più librerie anche di crittografia interna (AES), strutture dati e algoritmi per gestire tutte le operazioni del frontend tranne il chat bot che viene gestito da server Node.
